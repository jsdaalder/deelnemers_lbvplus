# Analysis (matching_ftm)

Reporting scripts for the FTM matching pipeline.

- `ftm/` – charts and exports using `pipelines/matching_ftm/data/processed/` as input; charts land in `ftm/charts/<YYYY_MM_DD>/` and exports go to `final_results/<YYYY_MM_DD>/`.
- `nrc/` – reserved for future cross-analysis of the NRC barn-level matches alongside FTM results (not yet added here).
